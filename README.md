# Quran Memorization Tracker

A simple, lightweight web-based tool to help track your Quran memorization (hifz) progress. Built with vanilla JavaScript and localStorage for offline functionality.

## Features

- Track memorization progress by surah and verse
- Add custom surahs with any number of verses
- Visual progress indicators with overall completion percentage
- Statistics tracking (total verses, memorized verses, completed surahs)
- Local storage persistence - your data stays on your device
- Mobile-friendly responsive design
- No external dependencies or frameworks

## Technologies

- Vanilla HTML5, CSS3, and JavaScript
- localStorage for data persistence
- Responsive design with CSS Grid and Flexbox
- Gradient backgrounds and modern UI

## Usage

1. Open `index.html` in any modern web browser
2. Your progress is automatically saved to localStorage
3. Add new surahs using the "Add Surah" button
4. Update memorized verses by typing in the input fields
5. Reset all progress with the "Reset Progress" button (confirmation required)

## Privacy

All data is stored locally in your browser's localStorage. No data is sent to any servers or third parties.
