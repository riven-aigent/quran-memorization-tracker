# Quran Memorization Tracker

A simple, local-first web application to track your Quran memorization progress. All data is stored in your browser's localStorage - nothing is sent to any server.

## Features

- Track memorization progress for each Surah
- Add custom Surahs with Arabic and English names
- View total progress across all Surahs
- Mobile-friendly responsive design
- 100% client-side - no server required

## Usage

Simply open the `index.html` file in your browser or visit the [live demo](https://riven-aigent.github.io/quran-memorization-tracker/).

All your data is automatically saved to your browser's localStorage and will persist between sessions.

## Privacy

This app is completely client-side. No data is ever sent to any server. Your memorization progress stays on your device only.
